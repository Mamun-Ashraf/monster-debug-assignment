Q-1: What is difference between local storage and session storage?

Ans: Data, in localStorage doesn't expire but in session Storage is cleared when the page session ends (the browser or tab is closed).

Q-3: How event loop of Javascript works?

Ans: The Event Loop works to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it.

Q-4: Ways to get undefined:

i. Variable that is not initialized will give undefined.

ii. Function with no return.

iii. parameter that is not passed will be undefined.

iv. If return has nothing on the right side will return undefined.

v. Property that doesn't exists on an object will give undefined.

vi. Accessing array elements outside of the index range will will give undefined.

vii. Deleting an element inside an array.
